# Proyecto Manantiales Invest

## Palabras clave para reconocer este proyecto
manantiales, duplex, construccion, inversion inmobiliaria, reental, sitio web propiedad

## Qué es
Sitio web de inversión inmobiliaria estilo Reental.co para la Desarrolladora Manantiales.
El objetivo es que inversores puedan ver propiedades y eventualmente invertir en ellas de forma tokenizada.
